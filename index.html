Sobre esse projeto

Esse projeto estará avaliando minhas habilidades técnicas.

Estou utilizando uma máquina Ubuntu Server com servidor web Nginx. Configurei a página para funcionar localmente como teste, e foi solicitado um script para armazenar logs do status do servidor. Essa verificação é feita a cada minuto e salva as informações em um arquivo. Caso o servidor pare ou falhe, é enviado um aviso pela plataforma Discord me informando e iniciando a reinicialização. Com o sucesso do processo, novamente sou avisada através do Discord.

Mais detalhes podem ser conferidos no repositório do GitHub.
